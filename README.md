##  Features

- Interactive financial charts
- Customizable chart options
- Real-time data updates _(planned)_

---

## Installation

### Clone the repository

```sh
git clone https://github.com/Paul-Lecomte/trading_graph.git
cd trading_graph
```

### Install dependencies

```sh
npm install
```

### Run the application

```sh
npm start
```
